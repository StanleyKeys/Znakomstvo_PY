Данный справочник имеет небольшой функционал:

Вкладка "Справочник":
"Добавить Запись" - Добавляет данные в текстовый файл в виде списка

"Экспорт в Excel" - переводит Имеющийся список телефонов (phonebook.txt) в формат *.xls

"Очистить Список" - очищает справочник

Вкладка "Поиск"

"Поиск" - ищет введеные пользователем данные в файле и выводит результаты в "Выпадающий список"(combobox)

"Показать" - при выборе результата из Combobox и нажатии данной кнопки появляется детальная информация по человеку




"Разработчики" - данные о создателе :)
